Troubleshooting

If tests fail:
  1. Check browser console (F12) for errors
  2. Verify all .js files exist in same directory
  3. Clear browser cache (Ctrl+Shift+R)
  4. Check file names match exactly (case-sensitive)
  5. See BUGFIX_INTEGRATION.md
